# 📊 Employee Performance Analysis (INX Future Inc)

## 📌 Project Overview

This project focuses on analyzing employee performance data to identify key factors that influence productivity, job satisfaction, and employee retention. Using machine learning techniques, this analysis provides actionable insights to support data-driven HR decision-making.

## 🛠 Tools & Technologies

* Python (Pandas, NumPy)
* Scikit-learn (Random Forest Regressor)
* Google Colab
* Data Visualization (Matplotlib / Seaborn)

## 🔍 Problem Statement

Organizations often struggle to:

* Identify key drivers of employee performance
* Reduce high attrition rates
* Optimize training and compensation strategies

This project aims to solve these problems through data analysis and predictive modeling.

## 🤖 Methodology

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Importance Analysis using Random Forest Regressor

## 📈 Key Insights

### 1. Employee Performance Drivers

* Job Satisfaction, Job Involvement, and Environment Satisfaction are the most influential factors in employee performance.

### 2. Attrition Analysis

* Departments such as Sales and Development show higher attrition rates.
* Indicates need for targeted retention strategies.

### 3. Training Impact

* Employees with higher training frequency tend to have better performance ratings.
* Suggests investment in training improves productivity.

### 4. Work-Life Balance

* Strong positive relationship with performance.
* Employees with better balance perform better.

### 5. Overtime Analysis

* Overtime does NOT significantly improve performance.
* Indicates risk of burnout and inefficiency.

### 6. Compensation Insights

* No significant gender pay gap found.
* Performance-based salary alignment is recommended.

## 💼 Business Impact

This analysis provides actionable recommendations for:

* Improving employee retention strategies
* Enhancing workforce productivity
* Designing effective HR policies
* Supporting data-driven decision making in organizations
